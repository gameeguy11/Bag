# Pickup Full Carryalls (Updated for Subnautica 2.0)

This is an **updated version** of the original *Pickup Full Carryalls* mod.  
It restores full functionality on Subnautica 2.0 while keeping the mod faithful to the original creator’s design.

The mod allows you to:

- Pick up Carry-alls and Waterproof Lockers even when they contain items  
- Open them directly from your inventory using Middle Mouse Button 
- Use them as portable storage without emptying them first  

---

## Features

- Pick up non‑empty Carry‑alls  
- Pick up non‑empty Waterproof Lockers  
- Open containers from your inventory  
- Middle‑click to access storage instantly   
- Updated for **Subnautica 2.0**, BepInEx, and Nautilus

---

## Installation

1. Install [BepInEx](https://www.openmods.net/mod/31?gameId=1)
2. Install [Nautilus](https://www.openmods.net/mod/30?gameId=1)
3. Download the latest release  
4. Extract to your Subnautica *BepInEx/plugins* folder

---

## Compatibility

- ✔ Subnautica 2.0+  
- ✔ BepInEx 5.x  
- ✔ Nautilus 1.x  
- ✔ Works with most inventory‑related mods
